# mondja
Monja blog with Django

# Note
Djangoを使ったシンプルなBlogアプリ

# Requirements
* Python==3.4.1
* Django==1.7.4
* django-dotenv==1.3.0
* dj-database-url==0.3.0
* gunicorn==19.2.1
* whitenoise==1.0.6

# Run
~~~
python manage.py runserver
~~~
Access to http://127.0.0.1:8000/
