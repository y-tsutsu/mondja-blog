# mondja
Monja blog with Django

# Note
Djangoを使ったシンプルなBlogアプリ（Python Tools for Visual Studio）

# Requirements
* python==3.4.2
* django==1.7.4
* django-dotenv==1.3.0
* dj-database-url==0.3.0
* gunicorn==19.2.1
* whitenoise==1.0.6
* psycopg2==2.5.2
* cloudinary==1.0.21
* pydenticon==0.2
* pytz==2014.10 

# Run
~~~
python manage.py runserver
~~~
Access to http://127.0.0.1:8000/
